230  * is able to follow the commands of the FRI client. The best possible value
231  * 1.0 is reached when the robot executes the given commands instantaneously.
232  * The tracking performance drops towards 0 when latencies are induced,
233  * e.g. when the commanded velocity, acceleration or jerk exceeds the
234  * capabilities of the robot.
