75  /**
76  * \brief Abstract FRI transformation client.
77  *
78  * A transformation client enables the user to send transformation matrices cyclically to the
79  * KUKA Sunrise controller for manipulating the transformations of dynamic frames in the
80  * Sunrise scenegraph.
81  * Usually, these matrices will be provided by external sensors.
82  * <br>
83  * Custom transformation clients have to be derived from this class and need to
84  * implement the provide() callback. This callback is called once by the
85  * client application whenever a new FRI message arrives.
86  *
87  * <b>This element is an undocumented internal feature. It is not intended to be used by applications as it might change or be removed in future versions.</b>
88  */

168  /**
169  * <br> <b>This element is an undocumented internal feature. It is not intended to be used by applications as it might change or be removed in future versions.</b> <br>
170  * \brief Provides a requested transformation matrix.
171  *
172  * A transformation matrix has 3x4 elements. It consists of a rotational matrix (3x3 elements)
173  * and a translational vector (3x1 elements). The complete transformation matrix has the
174  * following structure: <br>
175  * [Transformation(3x4)] = [Rotation(3x3) | Translation(3x1) ]
176  * <p>
177  * All provided transformation matrices need a timestamp that corresponds to their
178  * time of acquisiton. This timestamp must be synchronized to the timestamp
179  * provided by the KUKA Sunrise controller (see getTimestampSec(), getTimestampNanoSec()).
180  * <p>
181  * If an update to the last transformation is not yet available when the provide()
182  * callback is executed, the last transformation (including its timestamp) should be
183  * repeated until a new transformation is available.
184  *
185  * @throw FRIException Throws a FRIException if the maximum number of transformations is exceeded.
186  * @param transformationID Identifier string of the transformation matrix
187  * @param transformationMatrix Provided transformation matrix
188  * @param timeSec Timestamp encoded as Unix time (seconds)
189  * @param timeNanoSec Timestamp encoded as Unix time (nanoseconds part)
190  */
191  void setTransformation(const char* transformationID, const double transformationMatrix[3][4],
192  unsigned int timeSec, unsigned int timeNanoSec);
